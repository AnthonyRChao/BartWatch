# BartWatch 

## Description: 
- A full stack web application that displays real time information about BART (Bay Area Rapid Transit) to users.

## Data: 
- BART API (http://api.bart.gov/docs/overview/index.aspx)

## Technologies:
- Boostrap 
- Flask
- SQLite
